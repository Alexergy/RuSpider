# Статистика датасета RuSpider

## Общая информация
- **Всего примеров**: 8,034
- **Тренировочных**: 7,000
- **Валидационных**: 1,034

## Форматы файлов
- **JSON (упрощённый)**: для быстрой загрузки
- **JSON (полный)**: со всеми полями и токенами
- **CSV**: для анализа в таблицах
- **TXT**: человеко-читаемые примеры

## Структура данных
Каждый пример в полной версии содержит:
- id, db_id
- SQL запрос
- Вопрос на английском
- Вопрос на русском
- Описание на русском
- Уровень сложности
- Компоненты SQL (операции, таблицы, агрегации)
- Токены запроса и вопроса
